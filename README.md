# Bitcoin Cash Topup Releases

This repository provides binary APK releases and update metadata for **Bitcoin Cash Topup**.

Bitcoin Cash Topup is a non-production Android test app for merchants/shopkeepers who want to test Bitcoin Cash topup and printed voucher workflows.

## Current status

This app is currently in **non-production testing**.

Do not use this release for high-value funds, production merchant operations, or situations where loss of local app data would cause serious problems.

## What is published here

This repository may contain:

* Android APK release files
* SHA-256 checksum files
* release notes
* installation instructions
* update metadata for future in-app update checks

This repository does **not** contain the application source code.

## Installing

Download the latest APK from the GitHub Releases page.

Android may ask you to allow installation from your browser or file manager because this app is distributed outside Google Play.

Only install APKs from this official release repository.

## Verifying a download

Each APK release includes a SHA-256 checksum.

After downloading the APK, compare the SHA-256 checksum of your downloaded file with the checksum published in the release notes or `.sha256` file.

If the checksum does not match, do not install the APK.

## Data and update warning

Android should preserve local app data during normal updates when the package name, signing certificate, and version rules are unchanged.

However, this is still a non-production test release. Back up any important wallet recovery information before updating or reinstalling.
